01 / Web Design

パソコンスクール WEBサイト

HTML / CSS / Responsive

Case Study

パソコンスクール WEBサイト

初心者でも気軽に通えるパソコンスクールを想定し、 安心感と分かりやすさを大切に制作しました。

制作期間

2週間

ターゲット

初心者でも気軽に通いたい30代女性

担当範囲

デザイン / コーディング / レスポンシブ対応

Issue

課題

パソコンに苦手意識がある方にとって、スクールに通うことへの不安を減らし、 「自分でも学べそう」と感じてもらえるサイトにする必要がありました。

Point

工夫したこと

やさしい配色や丸みのあるデザインを取り入れ、親しみやすい印象にしました。 また、コース内容や受講までの流れを分かりやすく整理し、申し込みまで迷わず進める構成を意識しました。

Design

デザイン

メインカラーには安心感のあるブルーを使用し、アクセントカラーで明るさを加えました。 写真や余白を活かし、初心者でも安心して通える雰囲気が伝わるように制作しました。

Coding

コーディング

HTML/CSSを使用し、PC・スマートフォンのどちらでも見やすいレスポンシブ対応を行いました。 セクションごとの余白やカード配置を調整し、読みやすさと操作しやすさを意識しました。

02 / Web Design

privatesauna Website

Design / Coding

Case Study

TOTONOI プライベートサウナサイト

完全予約制プライベートサウナを想定し、 都会の喧騒から離れて心と身体を整えられる空間を表現した WEBサイトを制作しました。

制作時間

10時間

ターゲット

20代〜40代のサウナ愛好家・仕事終わりにリフレッシュしたい社会人

担当範囲

デザイン / コーディング / レスポンシブ対応

Issue

課題

サウナ施設の魅力だけでなく、 利用することで得られる「ととのう体験」や 非日常感を伝える必要がありました。 また、予約導線を分かりやすく設計し、 利用までスムーズに進められる構成を目指しました。

Point

工夫したこと

JavaScriptを使用したファーストビューのスライダーや スクロールアニメーションを実装し、 上質な空間を演出しました。 また、ロウリュやオロポ、アメニティなど、 サウナ体験全体が伝わるコンテンツ構成を意識しました。

Design

デザイン

ブラックを基調に木目や暖色系カラーを組み合わせることで、 高級感と温かみを表現しました。 写真を大きく使用し、施設の魅力や落ち着いた雰囲気が 直感的に伝わるデザインを目指しました。

Coding

コーディング

HTML/CSS/JavaScriptを使用し、 レスポンシブ対応を実施しました。 フェードアップアニメーション、 ローディング画面、 ハンバーガーメニュー、 予約フォーム・送信完了ページを実装し、 実案件を意識した構成で制作しました。

03 / No-Code Website

STUDIO制作 WEBサイト

STUDIO / Design / Responsive

Case Study

STUDIO制作 WEBサイト

STUDIOを使用して制作したWEBサイトです。 ノーコードでありながら、余白設計・情報整理・レスポンシブ対応を意識し、 見やすく伝わりやすいサイトを目指しました。

制作時間

約1週間

使用ツール

STUDIO / Figma / Photoshop

担当範囲

デザイン / STUDIO実装 / レスポンシブ対応

Issue

課題

ノーコード制作でもテンプレート感が出ないように、 余白・文字サイズ・画像配置を調整し、 サイト全体の世界観が伝わる構成にする必要がありました。

Point

工夫したこと

STUDIOの機能を活用しながら、PC・スマートフォンの両方で見やすくなるように調整しました。 また、CTAの配置やセクションごとの流れを意識し、最後まで見てもらいやすい構成にしました。

Design

デザイン

ターゲットに合わせて配色や写真の見せ方を調整し、 情報が伝わりやすいシンプルなデザインを意識しました。

No-Code

STUDIO実装

STUDIOを使用し、デザインから実装・公開まで行いました。 レスポンシブ時の余白や文字サイズを細かく調整し、 ノーコードでも完成度の高いWEBサイトになるよう制作しました。

About WEB Design

Design × Coding

Figmaを使用したデザイン制作から、 HTML/CSSによるコーディングまで一貫して行っています。 見た目の美しさだけでなく、 情報の伝わりやすさや使いやすさを意識した WEBサイト制作を心掛けています。
